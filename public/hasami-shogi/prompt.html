hasami-shogi プロンプト

HTML+CSS+バニラ JavaScript で 9x9 の「はさみ将棋」を実装してください。ファイルは index.html 1 枚にまとめ、style と script はインライン化してください。

はさみ将棋はルールに地域差があるため、必ず以下のルールセットに厳密に従って実装してください。

盤面と初期配置

  • 9x9 の将棋盤風の盤面
  • 人間（先手）は「歩」9 枚を最下段（自分から見て一番手前の行）に横一列に配置
  • CPU（後手）は「と」9 枚を最上段に横一列に配置
  • 駒はテキスト（「歩」「と」）で表示し、画像や絵文字は使わない

駒の動き

  • すべての駒は将棋の飛車と同じ動き: 縦または横に何マスでも移動できる
  • 他の駒（敵味方問わず）を飛び越えることはできない
  • 移動先は空きマスのみ（相手の駒があるマスには移動できない）

挟み取りのルール

  • 通常の挟み取り: 自分の駒を動かした結果、縦または横の直線上で、相手の駒（1 枚以上連続した列）の両端を自分の駒 2 枚で挟んだ場合、挟まれた相手の駒をすべて取る
  • 1 回の移動で縦と横、複数方向の挟みが同時に成立した場合は、成立したすべての駒を取る
  • 角の取り方: 盤の四隅にある相手の駒は、その駒に縦横で隣接する 2 マスの両方を自分の駒で塞ぐと取れる
  • 自分から挟まれた場合は取られない: 相手の駒 2 枚の間や、角で塞がれる位置に自分から移動しても、その駒は取られない。駒が取られるのは、相手の移動によって挟み・角塞ぎが成立した瞬間のみ

勝敗

  • 先に相手の駒を 5 枚取った側の勝ち
  • 自分の手番で合法手が 1 つもない場合は負け
  • 勝敗が決まったら結果を表示し、以降の操作を無効化する

UI 要件

  • 自分の駒をクリックすると選択状態になり、移動可能なマスをハイライト表示する
  • 選択中に別の自駒をクリックすると選択を切り替え、同じ駒をクリックすると選択解除
  • 双方の取った駒数を常時表示する
  • 手番表示（あなたの番 / CPU 思考中）
  • CPU は「取れる手を優先し、取られる手を避ける」ヒューリスティックで指す（強さは問わないが、無意味な自滅はしないこと）
  • リセットボタンあり
  • レスポンシブ（モバイル対応）
  • 絵文字は使わない

index.html 1 枚で完成させてください。
