# othello — プロンプト

このテーマの **凍結プロンプト**。新しいモデルで再現する時はこのまま渡す。

---

HTML+CSS+バニラJavaScript で 8x8 のオセロを実装してください。ファイルは `index.html` 1 枚にまとめ、style と script はインライン化してください。

## 要件

- 人間(黒) vs CPU(白)
- 初期配置は中央4マスに黒白2石ずつ
- 合法手はクリック前にハイライト表示
- 8方向の挟み込み判定で石を反転
- CPU は角優先・辺加点・反転数加点のヒューリスティック
- 合法手がなければ自動パス
- 両者パスまたは盤面が埋まったら終局して石数で勝敗表示
- リセットボタンあり
- レスポンシブ(モバイル対応)
- 絵文字は使わない

`index.html` 1 枚で完成させてください。
