mario-like-2d プロンプト

このテーマの凍結プロンプト。新しいモデルで再現する時はこのまま渡してください。

HTML+CSS+バニラ JavaScript で、スーパーマリオ風の 2D 横スクロールプラットフォーマー を実装してください。ファイルは index.html 1 枚にまとめ、style と script はインライン化してください。

任天堂の公式素材・名称・音楽・スプライトは使わないこと。オリジナルの見た目（図形・色面で十分）でよい。ゲームタイトルもオリジナル名にすること。

プレイ要件（必須）

  • 横スクロールの 2D プラットフォーマー（サイドビュー）
  • 左右移動: 矢印キー ←→ および A / D
  • ジャンプ: Space / 矢印キー ↑ / W
  • リスタート: R / 開始: Enter または Space
  • 重力・着地・穴落下ミス
  • 横歩き敵。上から踏むと撃破、横接触でダメージ
  • コイン（収集物）取得でスコア増加
  • ゴール到達でクリア表示
  • HUD にスコア常時表示、クリア/ゲームオーバー後に再開可能

ステージ要件

  • 画面幅より長いステージ（横スクロールが必要）
  • 足場の高低差が 1 段以上
  • 穴（落下死ゾーン）が 1 箇所以上
  • 敵 2 体以上、コイン 3 個以上

技術要件

  • 外部ライブラリ・CDN・画像・音声に依存しない（完全オフライン）
  • requestAnimationFrame 等でゲームループ
  • 絵文字は使わない
  • モバイル対応は必須ではない

index.html 1 枚で、「開始 → 走って跳んで敵を踏む → コイン取得 → ゴール」まで遊べる状態にしてください。説明・前置き・コードフェンスは不要です。
