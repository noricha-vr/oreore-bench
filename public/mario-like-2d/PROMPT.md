# mario-like-2d — プロンプト

このテーマの **凍結プロンプト**。新しいモデルで再現する時はこのまま渡してください。

---

HTML+CSS+バニラJavaScript で、スーパーマリオ風の **2D 横スクロールプラットフォーマー** を実装してください。ファイルは `index.html` 1 枚にまとめ、style と script はインライン化してください。

任天堂の公式素材・名称・音楽・スプライトは使わないこと。オリジナルの見た目（図形・色面で十分）でよい。ゲームタイトルもオリジナル名にすること。

## プレイ要件（必須）

- 横スクロールの 2D プラットフォーマー（サイドビュー）
- プレイヤー操作:
  - 左右移動: 矢印キー ←→ および A / D
  - ジャンプ: Space / 矢印キー ↑ / W
  - リスタート: R
  - 開始: Enter または Space（タイトルがある場合）
- 物理:
  - 重力がある
  - 地面・足場に着地できる（すり抜けしない）
  - 穴に落ちたらミス扱い
- 敵:
  - 横方向に歩く敵が 1 種類以上いる
  - **上から踏むと撃破**できる
  - 横から接触するとダメージ（残機減少 or ミス）
- コイン（または同等の収集物）を取るとスコアが増える
- ステージ右端（または明確なゴール地点）に到達するとクリア表示
- HUD にスコアを常時表示（残機やコイン数があればなお良い）
- タイトル画面またはスタート導線があり、クリア / ゲームオーバー後に再開できる

## ステージ要件

- 画面幅より長いステージ（横スクロールが必要になる長さ）
- 足場の高低差が 1 段以上ある
- 穴（落下死ゾーン）が 1 箇所以上ある
- 敵が 2 体以上配置されている
- コイン（収集物）が 3 個以上配置されている

## 技術要件

- Canvas または DOM のどちらでもよいが、ブラウザで `index.html` を開くだけで動くこと
- 外部ライブラリ・CDN・画像アセット・音声ファイルに依存しない（完全オフライン）
- `requestAnimationFrame` 等でゲームループを回す
- 絵文字は使わない
- モバイル対応は必須ではない（キーボード操作前提でよい）

## 完成条件

`index.html` 1 枚で、キーボードだけで「開始 → 走って跳んで敵を踏む → コイン取得 → ゴール」まで遊べる状態にしてください。説明文・前置き・コードフェンスは不要です。完成した HTML 本体だけを出力してください。
